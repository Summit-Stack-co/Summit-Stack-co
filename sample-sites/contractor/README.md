# Summit Ridge Roofing - Image Checklist

This checklist now matches the current contractor sample exactly.

The page uses:
- 1 full-width hero image
- 1 about/crew image
- 3 before/after image pairs

Use this checklist to source image assets for `sample-sites/contractor/`.

## Required Images

1. **Hero image**
   - **Purpose:** Full-width first impression behind the headline
   - **Content:** Roofing crew working or a house exterior with a strong roofline
   - **Recommended size:** `2400 x 1400` or larger
   - **Orientation:** Landscape
   - **Notes:** Use something that feels like Reno / Carson City area housing or a believable local jobsite. Avoid polished stock-photo energy.

2. **About section crew image**
   - **Purpose:** Trust / about section visual
   - **Content:** Crew working, setting ladders, inspecting shingles, measuring, or loading tools
   - **Recommended size:** `1800 x 1400`
   - **Orientation:** Landscape
   - **Notes:** Natural light, real clothing, real truck/tools, slightly imperfect is better than staged.

## Before / After Images

3. **Roof replacement before**
   - **Purpose:** First before/after example
   - **Recommended size:** `1600 x 1200`
   - **Notes:** Older shingles, visible wear, patching, or uneven roofline

4. **Roof replacement after**
   - **Purpose:** First before/after example
   - **Recommended size:** `1600 x 1200`
   - **Notes:** Same home and angle if possible

5. **Storm repair before**
   - **Purpose:** Second before/after example
   - **Recommended size:** `1600 x 1200`
   - **Notes:** Missing shingles, wind damage, or exposed problem area

6. **Storm repair after**
   - **Purpose:** Second before/after example
   - **Recommended size:** `1600 x 1200`
   - **Notes:** Same house, same framing, more uniform repaired area

7. **Gutter upgrade before**
   - **Purpose:** Third before/after example
   - **Recommended size:** `1600 x 1200`
   - **Notes:** Old or incomplete gutter/drainage edge

8. **Gutter upgrade after**
   - **Purpose:** Third before/after example
   - **Recommended size:** `1600 x 1200`
   - **Notes:** Cleaner install, same angle if possible

## Image Quality Standards

- No copyrighted manufacturer logos
- No fake award badges or sales overlays baked into the image
- Avoid overly glossy stock photos
- Real homes are better than commercial buildings for this page
- Before/after sets should match angle, framing, and lighting as closely as possible
- Natural local-looking images are better than dramatic or cinematic ones

## File Naming Suggestions

- `hero-roofing-crew.jpg`
- `about-crew-working.jpg`
- `before-roof-replacement.jpg`
- `after-roof-replacement.jpg`
- `before-storm-repair.jpg`
- `after-storm-repair.jpg`
- `before-gutter-upgrade.jpg`
- `after-gutter-upgrade.jpg`

## Implementation Notes

- Replace the hero image first
- Replace the about/crew image second
- Then swap in the three before/after image pairs
- If the hero photo is busy, keep a dark overlay so the white text stays readable
- Use `.jpg` or `.webp` for all photos
