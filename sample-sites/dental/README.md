# Willow Creek Dental & Aesthetics - Image Checklist

This checklist matches the current dental / med spa sample page.

The page uses:
- 1 soft hero background image
- 1 experience section image
- 5 gallery images

Use this checklist to source image assets for `sample-sites/dental/`.

## Required Images

1. **Hero background image**
   - **Purpose:** Soft visual behind the centered hero content
   - **Content:** Clean clinic interior or treatment room
   - **Recommended size:** `2400 x 1400` or larger
   - **Orientation:** Landscape
   - **Notes:** Bright, airy, neutral, and uncluttered

2. **Experience section image**
   - **Purpose:** Main supporting image in the split experience section
   - **Content:** Calm treatment room or patient room
   - **Recommended size:** `1800 x 1400`
   - **Orientation:** Landscape
   - **Notes:** Should feel quiet, clean, and comfortable

## Gallery Images

3. **Clinic interior**
   - **Purpose:** Gallery image
   - **Recommended size:** `1600 x 1200`

4. **Treatment room**
   - **Purpose:** Gallery image
   - **Recommended size:** `1600 x 1200`

5. **Equipment close-up**
   - **Purpose:** Gallery image
   - **Recommended size:** `1600 x 1200`

6. **Smiling patient**
   - **Purpose:** Gallery image
   - **Recommended size:** `1600 x 1200`
   - **Notes:** Natural expression, not overly staged

7. **Calm neutral aesthetic photo**
   - **Purpose:** Gallery image
   - **Recommended size:** `1600 x 1200`
   - **Notes:** Can be a waiting area detail, clean countertop, towels, mirrors, or similar clinic atmosphere shot

## Image Quality Standards

- No cheesy stock-photo smiles
- No harsh blue medical lighting
- Avoid crowded or cluttered rooms
- Keep tones soft, bright, and neutral
- Natural whites, beige, sage, and light gray will fit the page best
- Real clinic photography is better than overly polished ad imagery

## File Naming Suggestions

- `hero-clinic-interior.jpg`
- `experience-treatment-room.jpg`
- `gallery-clinic-interior.jpg`
- `gallery-treatment-room.jpg`
- `gallery-equipment-closeup.jpg`
- `gallery-smiling-patient.jpg`
- `gallery-calm-aesthetic.jpg`

## Implementation Notes

- Replace the hero image first
- Replace the experience section image second
- Then fill the gallery strip
- Use `.jpg` or `.webp` for all photos
- If the hero image is too busy, keep the soft overlay so the centered text stays readable
